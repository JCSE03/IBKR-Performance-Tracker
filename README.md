# IBKR Performance Tracker
I created this project to help better visualise the performances of various accounts on IBKR

# How to use
Step 1: Log into your IBKR investment account via the browser 

Step 2: Log in → Peforamnce & Reports → Statements → Activity Statement

Step 3: Period: Custom Date Range | From Date: Your Choice | To Date: Your Choice

Step 4: Download CSV

Step 5: Remove all sensitive information and replace with "REDACTED"

Step 6: Upload the CSV into this application

Step 7: Dashboard will display upong uploading.
